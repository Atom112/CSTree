## AI 发展简史

| 年代 | 里程碑 |
|:----:|--------|
| 1956 | 达特茅斯会议——AI 诞生 |
| 1960s | 符号推理、早期 NLP |
| 1970s | 第一次 AI 寒冬 |
| 1980s | 专家系统、反向传播 |
| 1990s | 统计学习、SVM |
| 2000s | 深度学习崛起（ImageNet 2012） |
| 2010s | Transformer、GAN、大模型 |
| 2020s | GPT、多模态、AI Agent |

## 小结

| 流派 | 核心思想 |
|:----:|---------|
| **符号主义** | 逻辑推理表示智能 |
| **连接主义** | 神经网络模拟大脑 |
| **行为主义** | 智能体与环境交互 |
