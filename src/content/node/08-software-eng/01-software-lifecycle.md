## 软件生命周期

| 阶段 | 说明 |
|:----:|------|
| **需求分析** | 确定要做什么 |
| **设计** | 确定怎么做 |
| **实现** | 编码 |
| **测试** | 验证正确性 |
| **部署** | 上线发布 |
| **维护** | 修 Bug、加功能 |

## 过程模型

| 模型 | 特点 | 适用 |
|:----:|:----:|:----:|
| **瀑布** | 线性顺序，阶段分明 | 需求明确的小项目 |
| **敏捷** | 迭代增量，持续反馈 | 需求变化快的项目 |
| **V 模型** | 测试对应开发各阶段 | 安全关键系统 |

**为什么先学这个？** 了解流程后，进入主流方法——[[agile-scrum|敏捷开发与 Scrum]]。
