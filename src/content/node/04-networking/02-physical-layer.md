## 比特如何在线路上"旅行"？

在网线上传输的并不是"0"和"1"的数字——而是电压的高低变化。

> 🏫 **类比：摩斯电码**
> 摩斯电码用短信号（·）和长信号（—）表示字母。物理层也一样——用不同形式的物理信号表示 0 和 1。

## 信号的编码方式

### 不归零编码（NRZ）

```
1 = 高电压（+5V）
0 = 低电压（0V）

时钟： █ █ █ █ █ █ █ █ █ █
数据： 1 0 1 1 0 1 0 0 1 0
NRZ：  ━┓ ━┓━┓ ━┓ ━ ━┓ ━
        ↑
        问题：连续多个 1 或 0——接收方不知道什么时候该采样
```

### 曼彻斯特编码

在每个比特中间都有电压翻转——时钟信号嵌入在数据中：

```
数据： 1    0    1    1    0
曼彻： ━┓  ┏┛  ━┓  ━┓  ┏┛
        ┛   ┓   ┛   ┛   ┓
       ↑比特中间翻转
```

> 💡 以太网早期使用曼彻斯特编码。现代高速以太网使用更复杂的编码（如 4B/5B 或 64B/66B）。

## 物理介质

| 介质 | 速度 | 距离 | 用途 |
|------|------|------|------|
| **双绞线（Cat 5e/6）** | 1-10 Gbps | 100m | 家庭/办公室以太网 |
| **同轴电缆** | 10-100 Mbps | 500m | 有线电视、早期以太网 |
| **光纤（单模）** | 10-100 Gbps | 40km+ | 骨干网、数据中心 |
| **光纤（多模）** | 1-10 Gbps | 550m | 数据中心内部 |
| **Wi-Fi** | 1-9.6 Gbps | 30-100m | 无线局域网 |
| **5G** | 1-10 Gbps | 数百米 | 移动网络 |

## 多路复用技术

如何让多个信号共用同一根线或同一频段？

| 技术 | 原理 | 例子 |
|------|------|------|
| **频分复用（FDM）** | 不同频率各占一段 | 广播电视、Wi-Fi 不同信道 |
| **时分复用（TDM）** | 轮流使用时间片 | 电话网络 |
| **波分复用（WDM）** | 不同波长的光 | 光纤通信 |
| **码分复用（CDMA）** | 不同编码序列 | 3G 移动网络 |

## 小结

| 概念 | 要点 |
|------|------|
| **物理层** | 将比特转换为物理信号 |
| **编码** | 0 和 1 映射为电压/光信号的变化 |
| **介质** | 双绞线、光纤、无线电——各有优劣 |
| **复用** | 多个信号共享同一传输介质 |
