## 四大范式

| 范式 | 核心思想 | 代表语言 |
|:----:|---------|---------|
| **过程式** | 程序 = 算法 + 数据结构 | C, Pascal |
| **面向对象** | 程序 = 对象 + 消息传递 | Java, C++, Python |
| **函数式** | 程序 = 函数组合 | Haskell, Scheme, Clojure |
| **逻辑式** | 程序 = 事实 + 规则 | Prolog |

## 小结

| 概念 | 要点 |
|:----:|------|
| **范式** | 编程的思维方式和组织风格 |
| **多范式** | 现代语言通常支持多种范式（Python、Scala） |
| **选择** | 不同范式适合不同问题类型 |

**为什么先学这个？** 了解范式后，深入学习[[variable-scope|变量、作用域与绑定]]。
