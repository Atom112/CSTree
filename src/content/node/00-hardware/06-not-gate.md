## 非门的真值表

非门（NOT Gate / Inverter）只有一个输入，**输出与输入相反**。

| 输入 | 输出 |
|------|------|
| 0    | 1    |
| 1    | 0    |

## 电路符号

```
输入 ──[1]── 输出
```

输出端常画一个小圆圈表示取反。

## 与非门和或非门

将非门与 [[and-gate|与门]] 或 [[or-gate|或门]] 结合，得到两种重要的复合门：

### 与非门（NAND）
与非门 = 与门 + 非门：两个输入都是 1 时输出 0，其他情况输出 1。

**NAND 是万能的**：只用与非门就能构造出所有其他逻辑门！

### 或非门（NOR）
或非门 = 或门 + 非门：两个输入都是 0 时输出 1，其他情况输出 0。

## 实际应用

- **信号反相**：需要反相信号时使用
- **振荡器**：奇数个非门首尾相连可以产生时钟脉冲
- **三态门**：非门用于控制使能信号

## 小结

非门是最简单的逻辑门，但它的取反能力是所有复杂电路的基础。结合 [[and-gate|与门]] 和 [[or-gate|或门]]，可以构造出任意复杂的数字电路。
