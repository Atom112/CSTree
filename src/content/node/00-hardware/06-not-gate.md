## 非门的真值表

与门和或门都有两个输入，而非门只有一个输入。它的工作极其简单：

> 我说"今天**不**放假"，你的心情就从"好"变成"差"。非门做的事就是这个——**把 1 变 0，把 0 变 1**。

| 输入 | 输出 |
|------|------|
| 0    | 1    |
| 1    | 0    |

这张表是整棵知识树中最短的——只有两行。因为一个输入只有两种可能，输入和输出永远相反。它也叫**反相器（Inverter）**。

## 电路符号

```mermaid
graph LR
    I[输入] --> NOT[非门 NOT]
    NOT --> O[输出]
```

在电路图中，非门常用输出端画一个**小圆圈**表示取反。这个小圆圈在后来的电路中会成为最常见的符号——你会在[[sr-latch|SR锁存器]]中看到两个与非门交叉连接，就是用与非门（AND + 小圆圈）实现的。

## 生活中的非门

- **宿舍楼的门禁灯**：门锁上（输入=1）→ 红灯亮（输出=1）？不——锁上时绿灯亮（取反）
- **电梯"已满"指示**：电梯内人数未满（输入=0）→ "可进"灯亮（输出=1）。人数满了（输入=1）→ 灯灭（输出=0）
- **Windows 关机**：你点"关机"（输入=1）→ 系统不执行"开机"（输出=0）

## 非门的晶体管实现

非门最简单——一个晶体管 + 一个上拉电阻：

```
电源(VCC) ──┬── 电阻 ──┬── 输出
            │          │
            │      ┌──┴──┐
            │      │ 晶体管 │  ← 输入 A
            │      └──┬──┘
            │         │
                 地(GND)
```

- A=1 → 晶体管导通 → 输出接地 → 输出=0
- A=0 → 晶体管断开 → 输出被电阻拉高 → 输出=VCC=1

## 三种基本门的对比

| 门 | 输入数 | 输出=1 的条件 | 类比 |
|:---:|:-----:|:-----------:|------|
| **与门** | 2 | A=1 **且** B=1 | 作业做完且房间整理好→能玩游戏 |
| **或门** | 2 | A=1 **或** B=1 | 红烧肉或糖醋排骨→去那个窗口 |
| **非门** | 1 | 输入=0（取反） | 不放假的"不"字，把好变差 |

这三种基本门是数字电路的"字母"——就像英文的 26 个字母可以组成任何单词。用它们可以组成[[half-adder|半加器]]、[[multiplexer|多路选择器]]，乃至整个 CPU。
