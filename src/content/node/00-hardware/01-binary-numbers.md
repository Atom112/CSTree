## 什么是二进制？

二进制（Binary）是一种**逢二进一**的记数系统，只使用两个数字：**0** 和 **1**。

与我们日常使用的十进制（0-9）不同，计算机内部的所有信息——数字、文字、图片、声音——最终都以二进制形式存储和处理。

## 为什么计算机用二进制？

1. **物理实现简单**：用电路的通/断表示 0/1，非常可靠
2. **抗干扰能力强**：只有两种状态，信号容易区分
3. **逻辑运算方便**：布尔代数可以直接用电路实现

## 二进制与十进制的转换

### 二进制 → 十进制

每位数字乘以 2 的幂次，然后相加：

```
1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰
      = 8 + 0 + 2 + 1
      = 11₁₀
```

### 十进制 → 二进制

不断除以 2，取余数，倒序排列：

```
13 ÷ 2 = 6 余 1
 6 ÷ 2 = 3 余 0
 3 ÷ 2 = 1 余 1
 1 ÷ 2 = 0 余 1

13₁₀ = 1101₂
```

## 位与字节

- **位（bit）**：二进制的一位，0 或 1
- **字节（byte）**：8 个 bit，计算机存储的最小可寻址单元

1 字节可以表示 2⁸ = 256 种不同的值（0-255）。
